# Change Log

All notable changes to this project are documented in this file.

## [0.1.0] - 2026-03-14

### Added
- Click on template name to apply the template
- Click on template item (when multi-select mode is on) to toggle selection
- Multi-select mode state is now persisted across sidebar sessions

## [0.0.1] - Initial release

- Initial release: save named include/exclude search templates, apply single or combined templates, persisted in workspace state.
